## 3. 免責テロップ

### 3-1. 概要欄常設（全動画共通・必須）

動画概要欄の最下部に以下を常設する。映像演出仕様書§12-2の規定文言そのまま。

```
──
⚠️ 免責事項
本チャンネルの動画は情報提供を目的としており、
金融商品の推奨・投資助言・医療行為の代替を目的としたものではありません。
投資判断・健康に関する判断はご自身の責任で行ってください。
詳しくは各分野の専門家にご相談ください。
```

### 3-2. 画面内ピンポイント表示（2箇所のみ）

YouTubeショート切り抜きされても免責が残るよう、特にセンシティブな2箇所のみ画面内に配置する。
文言は映像演出仕様書§12-1の規定をそのまま採用（独自アレンジ禁止）。

#### 箇所①：複利20%シミュレーション（19:00〜20:20頃）

**表示タイミング**：該当シーンの冒頭（19:00頃）
**表示秒数**：5秒程度
**文言**：

```
本動画は金融商品の推奨・投資助言を目的としたものではありません。
投資判断はご自身の責任で行ってください。
```

#### 箇所②：アーユルヴェーダ・宝石療法・薬草療法（24:00〜27:10頃）

**表示タイミング**：該当シーンの冒頭（24:00頃）
**表示秒数**：5秒程度
**文言**：

```
本動画の内容は情報提供を目的としており、医療行為・診断・治療の代替ではありません。
体調に不安がある方は医師にご相談ください。
```

## 4. 表示運用のチェックリスト

### 用語解説テロップ
- [ ] 全18項目の表示タイミングが該当発言と同期している
- [ ] 用語色＝ゴールド（#C5A059）、解説色＝ウォームホワイト（#FFFDF9）に統一
- [ ] 背景は半透明ディープモーブ帯で統一
- [ ] 強調テロップ表示中はフルテロップを非表示にしている

### 免責テロップ
- [ ] 概要欄最下部に§12-2の共通免責文が記載されている
- [ ] 複利20%シーン冒頭（19:00頃）に金融免責が表示されている
- [ ] アーユルヴェーダ冒頭（24:00頃）に健康免責が表示されている
- [ ] 免責文は§12-1の規定文言そのまま（独自アレンジなし）

### ブランド整合
- [ ] 禁止色（赤・黄・蛍光色）が使われていない
- [ ] ディープモーブはダーク背景としてのみ使用（テキスト色に使っていない）
- [ ] 用語解説テロップのフォントは明朝体・ウェイトMedium
- [ ] 免責テロップのフォントはゴシック体
